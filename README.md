📚 Repositório de Banco de Dados

Este repositório foi criado com o objetivo de armazenar e organizar todos os materiais produzidos durante as aulas de Banco de Dados.

Reunir em um único lugar conteúdos desenvolvidos ao longo do curso, facilitando:

Consulta de materiais
Revisão para provas e atividades
Compartilhamento de conhecimento

SQL
MySQL / PostgreSQL / SQLite
Ferramentas de modelagem (como brModelo, MySQL Workbench, etc.)
📖 Conteúdos Abordados

Alguns dos temas estudados:

Modelagem de dados (MER)
Modelo Relacional
Linguagem SQL (DDL, DML, DQL)
Normalização de dados
Chaves primárias e estrangeiras
Consultas e manipulação de dados
